  • Added pointer to private data in stmdev_ctx_t

  • add from_f16_to_f32 API
  • fix fifo_fsm_batch_set/get APIs

  • Fix fifo_batch_counter_threshold_set/get
  • Adding CODE_OF_CONDUCT.md and SECURITY.md
